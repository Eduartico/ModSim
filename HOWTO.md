# HOWTO

## Prerequisites

- Python: The project is developed using Python. Ensure Python (preferably Python 3.x) is installed.
- mesa
- numpy
- matplotlib
- pandas

```
pip install -r requirements.txt
```

## Running the Project

python proj/main.py

### Run visualization

In main.py file change iteractive_GUI to True: 

```
interactive_GUI = True
```
